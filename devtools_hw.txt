
Devtools HW

Зробити скрін відео виконання завдання.

 1. Відкрити Chrome DevTools
 2. Зайти в панель Elements.
 3. Знайти інспектором кнопку Submit і вирівняти слово Submit по центру
 4. Увімкнути режим мобільного деввайса (Device Mode)
 5. Виставити Dimensions: Responsive 
 6. Вибрати Responsive - Mobile
 7. Продемонструвати роботу дизайну Responsive
 8. Виставити Dimensions: Adaptive (select device).
 9. Створити свій девайс.
 10. Виставити Throttling low-end mobile
 11. оновити сторінку
 12. Перевести в ландшафтні режим.
 13. Вимкнути Device Mode
 14. Відкрити панель Console.
 15. Знайти 2 повідомлення Error у логах
 16. Зберегти ці 2 повідомлення лога як файл на комп'ютер.
 17. Знайти 1 повідомлення verbose
 18. Перегорнути логи Warnings, Info, User Messages.
 19. Відкрити вкладку Sources.
 20. Відкрити файл Index
 21. Відкрити вкладку Network
 22. В урлі відправити відправити http://162.55.220.72:5005/object_info_3?age=32&salary=1000&name=name
 23. Знайти лог request/responce  
 24. Відкрити вкладку Headers
 25. Переглянути General, Response Headers, Request Headers
 26. Знайти і скопіювати в новий текстовий файл header - User-Agent.
 27. Відкрити вкладку Preview
 28. Розгорнути всі рівні вкладеності JSON.
 29. Відкрити вкладку Application.
 30. Розгорнути всі Storage у лівому стовпчику.
 31. Зайти в Cache Storage і Application Cache
 32. Там скрізь буде порожньо.
 33. Відкрити будь-який сайт, який хочете. (Введіть урл сайту в адресний рядок і натисніть Enter)
 34. Відкрийте вкладку LightHouse
 35. Натисніть Generate report
 36. Встановити в браузер безкоштовний AdBlocker
 37. Установити в браузер Json Viewer
 38. Зайти на сайт https://ksendzov.com (https://ksendzov.com/)
 39. Зробити та зберегти скріншот видимої частини сторінки сайту.
 40. Зробити та зберегти скріншот усього сайту.
 41. Змінити локацію GPS. Продемонструвати на сайті Гугл карт.
 42. Змінити локацію GPS вдруге. Продемонструвати на сайті Гугл карт.
 43. Знайти інтерфейс для зміни User-Agent.
 44. Підставити інший User-Agent. 
 45. Надіслати запит на http://162.55.220.72:5005/object_info_3?age=32&salary=1000&name=name
 46. Перевірити, чи змінився ваш нативний User-Agent на новий, підставлений.
 47. Створити свій юзер-агент.
 48. Повторити пункти 45, 46.
 49. Знайти інтерфейс для зміни/додавання параметрів Throttling.
 50. Створити свої параметри поганого з'єднання і перевірити, як завантажуватиметься сторінка.
 51. Знайти інтерфейс для зміни/додавання параметрів Locations.
 52. Перевірити на Google карті як застосовуються зміни локації.
